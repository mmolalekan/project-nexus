# Qurbaan — Fullstack E-Commerce Project

Qurbaan streamlines the journey from farm to fork by providing a modern platform where users can purchase high-quality beef shares.
This repository contains the fullstack implementation of the project — **backend (Django)** and **frontend (Next.js)**.
